# 05 — Offres et formules

## Principe général

Les offres de Les Voyages de Cha doivent être claires, lisibles et rassurantes.

Elles doivent permettre au client de comprendre ce qu'il achète, ce qu'il reçoit, et ce qui reste à sa charge.

Règle à rappeler avec les offres :

Cha ne réserve pas à la place du client. Elle transmet les recommandations, liens, contacts et informations utiles. Le client réserve lui-même auprès des prestataires.

## Formule weekend : Évasion express — 2 à 5 jours

Cette formule concerne les courts séjours, week-ends, escapades ou mini-voyages.

Objectif :

- faire gagner du temps ;
- structurer une escapade ;
- proposer des adresses utiles ;
- donner un itinéraire clair ;
- éviter au client de se disperser dans les recherches.

### Découverte Essentielle

Contenu possible :

- recommandations transports ;
- recommandations hébergement ;
- itinéraire ;
- PDF récapitulatif.

### Aventure sur mesure

Contenu possible :

- recommandations transports ;
- recommandations hébergement ;
- itinéraire ;
- activités ;
- carte interactive ;
- carnet PDF ;
- Google Maps.

### Évasion Premium

Contenu possible :

- recommandations transports ;
- recommandations hébergement ;
- itinéraire ;
- location voiture ;
- activités ;
- restaurants ;
- conseils ;
- carnet de voyage PDF ;
- carnet de voyage édité ;
- Google Maps ;
- conseils personnalisés ;
- liste de choses à faire à moins de 10 minutes ;
- bonnes adresses.

## Formule séjour : Évasion + — plus de 5 jours

Cette formule concerne les séjours plus longs.

Elle reprend la logique de l'Évasion express, mais avec un niveau d'approfondissement plus important.

Objectif :

- construire un itinéraire plus complet ;
- organiser plusieurs étapes ;
- proposer une vision globale du voyage ;
- intégrer une carte interactive selon la formule ;
- proposer des recommandations adaptées au rythme du client.

## Tarifs repérés dans le document d'offre

Tarifs de travail à confirmer :

- Découverte Essentielle : 40 € ;
- Aventure sur mesure : 50 € ;
- Évasion Premium : 80 €.

Ces montants doivent être vérifiés avec le temps réel passé grâce au traceur de temps.

## Options

Options identifiées :

- option plongée : 5 € par jour plongé par personne ;
- supplément : 10 € au-delà de 3 étapes ;
- carte interactive enrichie ;
- carnet de voyage édité ;
- expériences locales ;
- danse ;
- aventures ;
- bonnes adresses spécifiques ;
- liste autour d'un hébergement ou d'un point précis.

## Livrables possibles

Selon la formule choisie, le client peut recevoir :

- PDF récapitulatif ;
- carnet de voyage PDF ;
- carnet de voyage édité ;
- carte interactive ;
- Google Maps ;
- itinéraire jour par jour ;
- sélection d'hébergements ;
- sélection d'activités ;
- sélection de restaurants ;
- contacts utiles ;
- liens à utiliser pour réserver ;
- conseils personnalisés ;
- points de vigilance ;
- alternatives.

## Limites de prestation à prévoir

Pour protéger le temps de Cha, chaque offre devra préciser :

- durée du voyage concernée ;
- nombre d'étapes incluses ;
- nombre de recherches incluses ;
- nombre de modifications incluses ;
- délai de livraison ;
- type de livrables inclus ;
- carte interactive incluse ou non ;
- niveau de détail ;
- suivi après livraison inclus ou non ;
- options payantes.

## Mention obligatoire à intégrer aux offres

Les prix, disponibilités, horaires et conditions des prestataires peuvent évoluer. Le client vérifie et réserve directement auprès des prestataires concernés.

## Objectif du traceur de temps

Le traceur de temps doit permettre de vérifier la rentabilité réelle de chaque offre.

Indicateurs utiles :

- temps prévu ;
- temps réel ;
- tarif facturé ;
- taux horaire réel ;
- rentabilité par formule ;
- ajustement éventuel du prix ou du périmètre.
