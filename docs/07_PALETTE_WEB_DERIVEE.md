# Note — Charte graphique et palette web dérivée

## Pourquoi cette note ?

La charte graphique de Les Voyages de Cha pose une base visuelle douce, élégante et reconnaissable.

Cependant, lors de l'adaptation de cette identité sur le site internet, il est apparu que l'application stricte des couleurs officielles pouvait donner un rendu un peu trop froid ou trop blanc à l'écran.

Pour conserver l'esprit de la marque tout en rendant le site plus chaleureux, une palette web dérivée a été créée.

## Palette officielle de la charte

Couleurs de référence :

- Bleu foncé : #204a55
- Bleu clair : #91cbdd
- Bleu / gris clair : #bbced0
- Beige clair : #efe8e1
- Noir : #000000
- Blanc / clair : #EDEDED

Ces couleurs restent la base de l'identité visuelle.

## Palette web dérivée

Pour le site, une déclinaison plus chaude a été ajoutée :

- Beige sable : #E8D7C6
- Beige chaud : #E6D3C1
- Crème lumineux : #F1E3D3
- Sable ombré : #DFC8B2
- Ivoire chaud : #F8ECDD

## Logique d'utilisation

La palette web dérivée ne remplace pas la charte.

Elle sert à adapter la charte à l'écran, pour obtenir un rendu :

- plus chaleureux ;
- plus doux ;
- plus immersif ;
- plus proche de l'univers voyage ;
- plus cohérent avec les visuels Instagram validés par Cha ;
- moins froid ;
- moins blanc ;
- moins institutionnel.

## Règle de cohérence

Le bleu profond reste utilisé pour :

- les titres ;
- les contrastes ;
- les repères visuels ;
- les boutons importants ;
- les éléments de structure.

Le bleu clair reste utilisé pour :

- les lignes graphiques ;
- les icônes ;
- les petits détails ;
- les respirations visuelles.

Le beige devient dominant pour :

- les fonds ;
- les cartes ;
- les sections ;
- l'ambiance générale du site.

## Formulation simple pour expliquer à Cha

La charte graphique est bien respectée, mais elle a été adaptée au web avec une déclinaison plus chaude des beiges.

L'objectif est de garder l'identité officielle tout en donnant au site une ambiance plus douce, plus chaleureuse et plus immersive, proche des visuels Instagram que Cha aime.

On ne change donc pas l'identité de marque : on la rend plus vivante à l'écran.

## Phrase de référence

La charte officielle donne l'identité.  
La palette web dérivée donne la chaleur et l'ambiance du site.
