# Référence projet — Les Voyages de Cha

## 1. Positionnement

Les Voyages de Cha est une activité indépendante de travel planning.

Cha conçoit des voyages sur mesure, immersifs et humains.  
Elle s'adresse à des voyageurs curieux, actifs, en quête de sens, qui veulent vivre une destination plutôt que la consommer.

Cha n'est pas une agence de voyage.  
Cha n'est pas un comparateur.  
Cha est la personne de confiance qui conçoit des voyages qui ont du sens.

Signature :
Voyagez autrement, vivez intensément.

## 2. Vision / manifeste

Le voyage est plus qu'un déplacement.

C'est une manière de découvrir, de ressentir, de créer des moments uniques et de transformer son regard sur le monde.

Valeurs centrales :
- authenticité ;
- partage ;
- bienveillance ;
- liberté ;
- émotion ;
- expertise terrain.

## 3. Territoire éditorial

Le territoire de Cha se situe entre :
- le voyage ;
- l'humain ;
- l'intensité de l'expérience.

Axes à privilégier :
- immersion culturelle ;
- rencontre avec les locaux ;
- expériences sportives, sensorielles et humaines ;
- voyage vécu de l'intérieur ;
- projection client ;
- expertise simple ;
- appels à l'action doux.

À éviter :
- jargon touristique ;
- listes froides ;
- discours institutionnel ;
- promesses irréalistes ;
- ton trop commercial ;
- tourisme de masse idéalisé.

## 4. Ton de voix

Le ton doit être :
- conversationnel ;
- chaleureux ;
- sincère ;
- motivant ;
- humain ;
- accessible ;
- parfois légèrement humoristique, jamais forcé.

Mots clés :
- immersion ;
- vivre ;
- ressentir ;
- intensément ;
- sur mesure ;
- autrement ;
- humain ;
- expérience.

Expressions utiles :
- je vous emmène ;
- l'idée, ce n'est pas de... mais de... ;
- ce que j'aime dans ce voyage ;
- ce genre de moment qu'on n'oublie pas.

## 5. Charte graphique

Couleurs officielles :
- bleu foncé : #204a55 ;
- bleu clair : #91cbdd ;
- bleu/gris clair : #bbced0 ;
- beige clair : #efe8e1 ;
- blanc : #ffffff ;
- noir : #000000.

Typographies :
- Poppins Regular : textes ;
- Poppins Semibold : sous-titres ;
- Poppins Bold : grands titres ;
- Poppins Italic : annotations ;
- Back to back : signature, accents manuscrits, éléments émotionnels.

Direction visuelle :
- claire ;
- lumineuse ;
- organique ;
- douce ;
- inspirée bord de mer / sable / lumière naturelle ;
- pas bling-bling ;
- pas corporate ;
- pas de gros blocs agressifs ;
- pas de marron trop lourd ;
- pas d'effet masculin / cabinet conseil.

Le site doit se rapprocher de l'ambiance des templates Instagram :
- beaucoup d'air ;
- fonds clairs ;
- titres Poppins forts mais maîtrisés ;
- manuscrit utilisé en accent ;
- traits fins bleu clair ;
- logo lisible ;
- composition élégante et légère.

## 6. Offres

Offre 1 :
Évasion express  
Pour les week-ends et escapades courtes, environ 2 à 5 jours.

Offre 2 :
Évasion +  
Pour les séjours de 5 jours et plus.

Options :
- plongée ;
- étapes supplémentaires ;
- expériences locales ;
- carte interactive selon formule ;
- carnet PDF ;
- conseils personnalisés.

## 7. Règle métier importante

Cha ne réserve pas à la place du client.

Cha conçoit, organise, structure et transmet :
- itinéraire ;
- liens ;
- contacts ;
- adresses ;
- recommandations ;
- activités ;
- repères utiles.

Le client effectue directement ses réservations, paiements, confirmations, modifications et annulations auprès des prestataires concernés.

Formulation à privilégier :
Cha conçoit votre projet de voyage et vous transmet tout pour réserver sereinement.

## 8. Bureau de Cha

Le Bureau de Cha est privé.

Il ne doit pas apparaître dans le menu public.

Accès validé :
- uniquement en cliquant sur le logo en haut à gauche.

## 9. Prochaine reprise du site

Avant de modifier encore le design :
1. nettoyer le CSS ;
2. supprimer les surcouches contradictoires ;
3. repartir de la charte graphique officielle ;
4. refaire la section Offres avec une composition plus légère, plus féminine et plus proche des templates Instagram ;
5. éviter les gros blocs commerciaux.
