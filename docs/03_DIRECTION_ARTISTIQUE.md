# 03 — Direction artistique

## Inspiration validée

L'esprit visuel validé pour Les Voyages de Cha est celui des templates Instagram doux, chauds et élégants présentés à Cha.

Cette inspiration plaît à Cha et doit servir de référence pour le site.

## Ambiance recherchée

- beige dominant ;
- bleu profond en accent ;
- bleu clair en respiration ;
- lumière naturelle ;
- bord de mer ;
- Méditerranée ;
- voyage sensible ;
- chaleur ;
- douceur ;
- émotion ;
- humain ;
- premium accessible ;
- jamais froid ;
- jamais corporate ;
- jamais bling-bling.

## Palette d'inspiration

- Blanc : #FFFFFF

## Palette officielle issue de la charte graphique

- Bleu foncé : #204a55
- Bleu clair : #91cbdd
- Bleu/gris clair : #bbced0
- Beige clair : #efe8e1
- Blanc / clair : #EDEDED
- Noir : #000000

## Règle couleur

Le beige doit être très présent.

Le bleu foncé sert à structurer, hiérarchiser et contraster.

Le bleu clair sert à apporter de la fraîcheur, de la respiration et des touches lumineuses.

Le blanc sert à alléger l'ensemble.

Le site ne doit pas être dominé par un bleu froid.

## Typographies

Esprit typographique :

- titres : Poppins Bold ou équivalent ;
- sous-titres : Poppins Semibold ou équivalent ;
- textes : Poppins Regular ou équivalent ;
- accents émotionnels : typographie manuscrite type Back to Black.

La typographie manuscrite doit être utilisée avec parcimonie, comme accent émotionnel, pas partout.

## Images

Les images doivent évoquer :

- bord de mer ;
- lumière naturelle ;
- ruelles méditerranéennes ;
- horizon ;
- voyage sensible ;
- scènes de vie ;
- carnets ;
- cartes ;
- mer ;
- coucher de soleil ;
- immersion ;
- projection client ;
- douceur ;
- chaleur ;
- aventure accessible.

Les images ne doivent pas seulement illustrer une destination. Elles doivent faire ressentir une émotion et donner envie de se projeter.

## Phrases d'esprit

- Tu n'as pas besoin d'aller loin pour te sentir ailleurs.
- Voyagez autrement, vivez intensément.
- Et si ton prochain voyage te ressemblait vraiment ?
- Des voyages qui se vivent.
- Un voyage conçu pour toi.
- On en parle ?

## À appliquer au site

À renforcer :

- grands aplats beiges ;
- sections aérées ;
- cartes douces ;
- ombres légères ;
- images de voyage intégrées ;
- petits traits graphiques bleu clair ;
- icônes fines ;
- accents manuscrits ;
- boutons arrondis ;
- ton chaleureux.

À éviter :

- site froid ;
- bleu trop dominant ;
- fond clinique ;
- rendu administratif ;
- blocs trop techniques ;
- surcharge de texte ;
- esthétique agence de voyage catalogue ;
- manque de chaleur.

## Direction finale

Le site Les Voyages de Cha doit donner la sensation suivante :

J'arrive dans un univers doux, clair, beau, humain et rassurant. Je comprends que Cha va m'aider à créer un voyage qui me ressemble, sans me vendre une formule impersonnelle.
